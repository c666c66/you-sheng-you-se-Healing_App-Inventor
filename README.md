## 關於 APP ##
- 透過 APP 物聯網 (IoT) 架構設計，呈現結合「聽覺、視覺、嗅覺」之創新情境設計，以提供一種可擴充式的 APP 療癒應用工具。
- APP 可播放療癒系音樂，並且可操控燈泡、精油，使聽覺、視覺、嗅覺皆可感受療癒。
